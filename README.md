# Platform Scripts

Bash scripts for platform engineering and infrastructure automation.
Built as part of my Platform Engineer roadmap (July 2 – Dec 31, 2026).

## Scripts
- healthcheck.sh — HTTP health check with retries
- log-rotate.sh — Archive and compress old logs
- deploy.sh — Deployment health checks
- backup-db.sh — Database backup (PostgreSQL)
- monitor-disk.sh — Alert on high disk usage
